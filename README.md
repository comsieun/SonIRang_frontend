# SonIRang_frontend

# React

React와 styled-components로 만든 간단한 웹 애플리케이션입니다.  
프로젝트는 Create React App(CRA)으로 시작되었습니다.

## 📁 프로젝트 구조

/ 👉 메인 페이지 (MainPage)
/meeting 👉 미팅 페이지 (MeetingPage)

## 🛠 사용 기술

- React
- React Router DOM
- styled-components
- Create React App

## 🚀 실행 방법

1. 의존성 설치
npm install

2. 개발 서버 실행
npm start
